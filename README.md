# GameOfLifeEditor
An interactive application that allows the user to draw and run configurations in Conway's Game of Life.

Unfortunately, I have lost the original source code for this application, but the executable still exists.

To begin, run GOLedit.exe.  The program will show a grid of cells.  Controls will appear in the console, but I've listed them below as well.

Controls:
-To activate a cell, left click.
-To deactivate a cell, right click.
-To clear all cells, press "c".
-To begin simulation, press "Enter".
-To pause the simulation, press "Space".
-To increment by one frame, press "Right Arrow".
-To quit, press "Escape".

Have fun experimenting with Conway's Game of Life!
